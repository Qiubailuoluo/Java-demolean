## 核心概念

### 定义

面向对象将数据与行为封装在对象中。多态通过父类引用调用子类实现，运行时绑定方法。

### 常见误区

- 滥用继承而非组合
- 重写 `equals` 却不重写 `hashCode`
- 接口与抽象类职责混淆（接口偏契约，抽象类偏共享实现）

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `class` / `interface` / `abstract` | 类型定义 |
| `extends` / `implements` | 继承与实现 |
| `@Override` | 标记重写 |
| `record`（Java 16+） | 不可变数据载体 |
