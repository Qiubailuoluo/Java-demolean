## 核心概念

### 定义

泛型在编译期约束类型，避免强制转换与 `ClassCastException`。运行时擦除为原始类型或边界类型。

### 常见误区

- 创建泛型数组 `new T[10]`
- 混用 `? extends` 与添加元素
- 认为运行时能获取 `List<String>` 的 `Class`

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `<T>` | 类型参数 |
| `? extends T` | 上界通配符（只读倾向） |
| `? super T` | 下界通配符（只写倾向） |
