## 核心概念

### 定义

LTS 版本（8、11、17、21）是企业常用基线。本仓库统一 **Java 17**，涵盖文本块、switch 表达式、`record`、`sealed` 等。

### 常见误区

- 认为 `var` 是动态类型（仍是静态类型推断）
- `record` 可随意添加可变字段（应为不可变组件）
- 混用旧版 API 与新 API 无统一风格

## 关键 API / 注解

| 版本 | 特性 |
|------|------|
| 8 | Lambda、Stream、Optional |
| 9～11 | 模块、var（10 局部变量） |
| 14+ | switch 表达式、文本块 |
| 16+ | `record` |
| 17 | `sealed class` |
