## 核心概念

### 定义

`java.util` 集合框架提供统一接口操作一组对象。`List` 有序可重复；`Set` 不重复；`Map` 键值对。

### 常见误区

- 在遍历时直接 `list.remove(i)` 导致并发修改
- 将可变对象作 `HashMap` 的 key 且修改参与 hash 的字段
- 认为 `HashSet` 保证顺序

## 关键 API / 注解

| 名称 | 说明 |
|------|------|

| `HashSet` / `TreeSet` | 集合实现 |
| `HashMap` / `LinkedHashMap` | 映射实现 |
