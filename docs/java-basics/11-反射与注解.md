## 核心概念

### 定义

反射在运行时检查与操作类成员。注解为代码元数据，配合反射可实现简单框架行为（Spring 即基于此扩展）。

### 常见误区

- 过度使用反射替代正常多态
- 注解未指定 `@Retention(RUNTIME)` 导致反射读不到
- 破坏访问控制后依赖 `setAccessible(true)` 无审计

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `Class.forName` / `getDeclaredMethods` | 反射入口 |
| `@interface` | 自定义注解 |
| `@Retention` / `@Target` | 元注解 |
