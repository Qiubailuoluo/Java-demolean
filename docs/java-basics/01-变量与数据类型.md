## 核心概念

### 定义

Java 变量需先声明类型再使用。基本类型直接存值；引用类型存对象地址。`String` 为引用类型但行为类似不可变值对象。

### 常见误区

- 用 `==` 比较字符串内容（应使用 `equals`）
- 忽视 `int` 与 `Integer` 在 null 场景下的 NPE
- 认为 `float`/`double` 可精确表示所有小数

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `byte/short/int/long` | 整数类型 |
| `float/double` | 浮点 |
| `char/boolean` | 字符与布尔 |
| `Integer.valueOf` | 包装类创建 |
| `String.equals` | 内容比较 |

## 代码片段

```java
public class VariableDemo {
    public static void main(String[] args) {
        int age = 25;
        double price = 19.99;
        String name = "JavaLean";
        System.out.println(name + " age=" + age + " price=" + price);
        Integer boxed = Integer.valueOf(age);
        System.out.println("boxed: " + boxed);
    }
}
```
