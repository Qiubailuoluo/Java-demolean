## 核心概念

### 定义

流程控制决定语句执行顺序。Java 17 支持 `switch` 表达式与 `yield`，可减少样板代码。

### 常见误区

- `switch` 对 `String` 忘记 `break` 导致贯穿（传统 switch）
- 在增强 for 中修改集合结构导致 `ConcurrentModificationException`
- 浮点数用 `==` 判断相等

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `switch` / `case` / `default` | 多分支 |
| `yield` | switch 表达式返回值 |
| `break` / `continue` | 跳出循环或本次迭代 |
