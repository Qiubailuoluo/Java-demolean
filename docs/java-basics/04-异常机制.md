## 核心概念

### 定义

异常是程序运行时错误的对象表示。Checked 异常必须处理或声明；`RuntimeException` 及其子类为 Unchecked。

### 常见误区

- 捕获 `Exception` 后空处理（吞异常）
- 在 `finally` 中 `return` 覆盖 try 返回值
- 用异常控制正常业务流程

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `Throwable` / `Exception` / `RuntimeException` | 层次 |
| `try-with-resources` | 自动关闭 `AutoCloseable` |
| `throws` | 声明 Checked 异常 |

## 代码片段

```java
try (var reader = new java.io.StringReader("ok")) {
    // 使用资源
} catch (IOException e) {
    throw new IllegalStateException("read failed", e);
}
```
