# 综合演练：迷你电商

## 元信息

| 项 | 值 |
|----|-----|
| 板块 | springcloud |
| 难度 | 进阶 |
| 预估时长 | 6 小时 |
| 前置 | 02～08 篇及 [_labs/mini-mall-lab.md](_labs/mini-mall-lab.md) |

## 学习目标

- [ ] 串联 Eureka、Feign、Gateway 完成端到端调用
- [ ] 按端口表启动多进程无冲突
- [ ] 可选接入 Config、Zipkin、Resilience4j
- [ ] 总结微服务拆分与联调经验

## 场景说明

| 服务 | moduleId | 职责 |
|------|----------|------|
| 注册中心 | sc-eureka-server | 服务发现 |
| 商品 | sc-feign-provider | 商品查询 |
| 订单 | sc-feign-consumer | 下单聚合商品信息 |
| 网关 | sc-gateway | 对外统一 API |

详细步骤见 **[迷你电商综合演练](_labs/mini-mall-lab.md)**。

## 子项目规格（组合，非新工程）

本知识点**不新建** `moduleId`，复用已有模块：

| 步骤 | 模块 | 动作 |
|------|------|------|
| 1 | sc-eureka-server | 启动 8761 |
| 2 | sc-feign-provider | 注册为 `product-service` |
| 3 | sc-feign-consumer | Feign 调商品，暴露 `/orders` |
| 4 | sc-gateway | 路由 `/mall/**` 到下游 |

### 可选增强

| 增强 | 模块 |
|------|------|
| 统一配置 | sc-config-server + 各客户端 import |
| 链路 | sc-sleuth-zipkin + Gateway |
| 熔断 | consumer 侧 Resilience4j 或 Feign fallback |
| 消息 | 下单后 `sc-rabbitmq-demo` 发订单创建事件 |

## 验收清单

- [ ] `curl http://localhost:9000/mall/products/1` 返回商品 JSON
- [ ] `POST` 创建订单返回含商品名的订单 DTO
- [ ] Eureka 控制台可见全部实例 UP
- [ ] （可选）Zipkin 可见跨 Gateway trace

## 练习

1. 为订单接口添加参数校验与全局异常（复用 Boot 08 经验）。
