## 核心概念

### 定义

Spring Boot 约定优于配置，通过 starter 依赖与条件化自动配置快速搭建应用。`@SpringBootApplication` = `@Configuration` + `@EnableAutoConfiguration` + `@ComponentScan`。

### 常见误区

- 认为 Boot 取代 Spring 核心（Boot 是构建与自动配置层）
- 组件扫描路径未包含自定义包导致 Bean 未注册
- 盲目排除自动配置却不清楚影响

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `@SpringBootApplication` | 启动类注解 |
| `@EnableAutoConfiguration` | 启用自动配置 |
| `SpringApplication.run` | 启动入口 |

### Maven 依赖

```xml
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
</dependency>
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-test</artifactId>
  <scope>test</scope>
</dependency>
```

POM 基线见 [_pom-baseline.md](_pom-baseline.md)，`artifactId` 设为 `sb-01-starter`。

### 包结构

| 类 | 说明 |
|----|------|
| `StarterApplication` | 启动类 |
| `HealthController` | `GET /ping` 返回 `pong` |

### application.yml

```yaml
server:
  port: 8080
spring:
  application:
    name: sb-01-starter
```

### 运行与验证

```bash
cd springboot/sb-01-starter
mvn spring-boot:run
curl http://localhost:8080/ping
```

**期望：** `pong`
