- [ ] 能排除静态资源路径

## 核心概念

### 定义

Filter 作用于 Servlet 层；Interceptor 在 Controller 前后。鉴权、日志、链路 ID 常见于此。

### 常见误区

- 在 Interceptor 中写响应后仍调用 `preHandle` 返回 true
- 忘记 `WebMvcConfigurer.addInterceptors` 注册
- 与 Spring Security 过滤器链混淆

## 关键 API / 注解

| 名称 | 说明 |
|------|------|
| `HandlerInterceptor` | MVC 拦截器 |
| `WebMvcConfigurer` | 注册拦截器 |
| `Filter` / `@WebFilter` | Servlet 过滤器 |
